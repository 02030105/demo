# 儿童识字小报生成器

一个专为5-9岁儿童设计的智能学习材料生成工具，使用AI生成带有中文和拼音标注的场景化识字小报。

## 项目结构

```
识字小报生成器/
├── server.js              # Express服务器主文件
├── package.json           # 项目配置和依赖
├── README.md              # 项目说明文档
├── src/                   # 源码目录（预留）
├── public/                # 静态资源目录
│   └── index.html         # 主页面HTML文件
└── node_modules/          # npm包目录（自动生成，不上传）
```

## 功能特点

- 🎨 **场景化生成**：支持超市、医院、公园、学校、动物园等多种场景
- 📚 **智能标注**：自动为生成的小报添加中文和拼音标注
- 🔧 **易于使用**：简洁的Web界面，输入API密钥即可使用
- 💾 **一键保存**：支持生成图片的直接下载
- 🎯 **专为儿童设计**：适合5-9岁儿童的认知和学习特点

## 快速开始

### 1. 安装依赖

```bash
cd 识字小报生成器
npm install
```

### 2. 启动服务器

```bash
npm start
```

或使用开发模式（自动重启）：

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问：`http://localhost:3000`

## 使用说明

### 获取API密钥

1. 访问 [KIE AI API密钥页面](https://kie.ai/api-key)
2. 注册并获取Nano Banana Pro的API密钥
3. 在应用中输入API密钥

### 生成小报

1. 输入API密钥
2. 选择场景（超市、医院、公园、学校、动物园）
3. 输入小报标题（如：快乐超市、有趣的动物园）
4. 点击"生成小报"按钮
5. 等待生成完成（约30-60秒）
6. 查看并下载生成的小报

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **后端**：Node.js, Express.js
- **AI服务**：KIE AI Nano Banana Pro

## 开发说明

### 项目依赖

- **express** (4.18.2): Web服务器框架
- **cors** (2.8.5): 跨域资源共享支持
- **nodemon** (3.0.1): 开发时自动重启服务（仅开发环境）

### API集成

项目使用KIE AI的Nano Banana Pro模型生成图片，主要API端点：

- 创建任务：`POST https://api.kie.ai/api/v1/jobs/createTask`
- 查询任务：`GET https://api.kie.ai/api/v1/jobs/recordInfo`

### 调试技巧

1. 打开浏览器开发者工具（F12）
2. 查看Console标签的日志输出
3. 观察任务创建和轮询过程的详细信息

## 注意事项

- API密钥会保存在浏览器的localStorage中
- 每次生成需要消耗API配额，请合理使用
- 生成时间约30-60秒，请耐心等待
- 如果图片无法显示，会提供备用方案（在新窗口打开或下载）

## 故障排除

### 问题：图片无法显示

**解决方案：**
- 查看控制台日志，检查是否有CORS错误
- 使用页面提供的"在新窗口打开图片"或"下载图片"功能
- 确保网络连接正常

### 问题：API调用失败

**解决方案：**
- 检查API密钥是否正确
- 确认账户余额充足
- 查看网络是否正常
- 检查控制台的错误信息
